# PetHotel — Система учёта клиентов гостиницы для животных

## Что это?
Веб-приложение для автоматизации работы гостиницы для животных. 
Система позволяет вести учёт клиентов и их питомцев, управлять бронированиями, формировать отчёты и разграничивать доступ по ролям (администратор, менеджер, клиент).

## Стек

- **Backend:** Python 3.11, FastAPI, SQLAlchemy
- **База данных:** PostgreSQL
- **Frontend:** HTML5, CSS3, JavaScript
- **Аутентификация:** JWT

## Как запустить?
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --port 8000

##ER - БД

**Всего таблиц: 9**

| Таблица | Назначение |
|---------|------------|
| `roles` | Роли пользователей |
| `users` | Учётные записи |
| `clients` | Владельцы животных |
| `animals` | Питомцы |
| `bookings` | Бронирования |
| `services` | Услуги |
| `bookings_services` | Связь бронирований и услуг |
| `invoices` | Счета на оплату |
| `audit_log` | Журнал аудита |
